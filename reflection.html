Responsive web design illustrates a way of

web design that allows websites and pages to display on all

screen or devices by automatically adapting to the screen,

such as laptop, tablet or mobile phone.

Responsive Web Design helps accomplish

that one site could respond differently when viewed on

devices of different sizes, which could be more efficient

for both designers and developers.

Responsive web design achieves the adaption

for the layout to different screens by using fluid, proportion-

based grids, flexible images, and CSS3 media queries.

In detail, the fluid grid helps to illustrate page

elements in different percentages depends on different screens

or devices. Significantly, CSS3 media queries allow the page

to use plenty of CSS style rules for characteristics to display

pages on screens.

As a saying about Reponsive Web Design,

“Content is like water”, which helps users or visitors

navigate the website easily and efficiently.

During the process of finishing “storefront.html” and

“reflection.html”, I used fluid grids, which illustrated in

structure properly and clearly. Additionally, several CSS rules

have been set to achieve flexibility for the website to be

displayed on different devices or screens.

For instance, to improve pages, I designed “dropdown” to

illustrate specific description for each selected product, which

could not only enhance the page to be more neat and clear

to understand, but also improve the communication and

interactivity between visitors and websites.
